# AgenticMonkey v3 — COMPLETE REDESIGN BRIEF

## REFERENCE: labs.chaingpt.org
- MASSIVE display typography that fills the viewport
- 3D mascot character as the HERO centerpiece
- Light/white background (NOT dark mode)
- Minimal copy, maximum visual impact
- Partner/client logos prominently displayed
- Playful, bold, memorable brand identity
- Numbered services with clean layout
- Testimonials section
- Team section

## DESIGN PHILOSOPHY
- The MONKEY MASCOT is the brand. It should be everywhere.
- Typography should be HUGE and bold — think magazine/editorial style
- Break the grid — asymmetric layouts, overlapping elements
- NOT a dark SaaS template. This needs personality.
- Every section should feel different and interesting
- Animations should be purposeful, not just "fade in from bottom"

## COLOR SCHEME (NEW)
- Primary background: #FAFAFA (warm white/off-white)
- Secondary background: #0A0A0F (deep black for contrast sections)
- Accent: #00D4FF (cyan) — keep this
- Secondary accent: #FF6B2C (warm orange for CTAs — like ChainGPT's orange)
- Text: #1A1A2E (near-black)
- Muted text: #6B7280

## TYPOGRAPHY
- Display/Hero: Space Grotesk, 80-120px, ultra bold, uppercase
- Body: DM Sans
- Make the typography THE design element

## LAYOUT — Section by Section

### 1. HERO (Full viewport)
- MASSIVE text: "AGENTIC MONKEY" split across the screen in huge display font
- Subtitle underneath: "BUILDING TOMORROW" or "ENGINEERING AUTONOMY"
- The robot monkey mascot IMAGE centered/overlapping the text
- Single orange CTA button: "Book a Strategy Call"
- Minimal — just the logo, text, mascot, and one button
- Light background

### 2. PARTNER/CLIENT LOGOS BAR
- Horizontal scrolling marquee
- "Powered by:" then logos (OpenAI, Anthropic, Google, etc — AI tools we use)
- Clean, grayscale logos

### 3. SERVICES (Numbered list, editorial style)
- "WHAT WE BUILD" in large text
- Two-column layout: left = big number (01, 02, etc), right = service title + short description
- Category pills at top: AI, Automation, Development, Consulting
- 6 services listed cleanly
- Dark background section for contrast

### 4. MASCOT SHOWCASE / ABOUT
- Full-width section with the monkey mascot large
- "Meet your AI implementation partner"
- Brief about text
- This section should feel like a character introduction

### 5. HOW IT WORKS
- 3 steps, big numbers, clean layout
- Discovery → Build → Deploy
- Each step has a short description

### 6. USE CASES / INDUSTRIES
- Grid of industry cards
- Each with an icon and brief description
- 5 industries

### 7. WHY CHOOSE US
- 4 differentiators in a unique layout (not just cards)
- Bold statements

### 8. TESTIMONIALS (placeholder for now)
- Carousel or grid of quotes
- Placeholder quotes from happy clients

### 9. CTA SECTION
- Bold "Ready to build?" or similar
- Orange CTA button
- Monkey mascot peeking in from the side

### 10. FOOTER
- Clean, minimal
- Links, contact, copyright
